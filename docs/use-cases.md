# Cas d'usage

## UC-1 — Explorer un organe

> Un visiteur veut comprendre quelles maladies sont liées au cœur.

1. Arrivée sur l'accueil.
2. Clique « Explorer le corps ».
3. Survole le cœur → tooltip + halo lumineux.
4. Clique → ouvre `/corps/coeur` avec stats, fonctions, sous-structures, maladies liées.
5. Clique sur « Infarctus du myocarde » → ouvre `/maladies/infarctus-myocarde`.

## UC-2 — Comparer deux maladies

> Un étudiant veut comparer le diabète et l'obésité.

1. Va dans `/maladies`.
2. Filtre par catégorie « Métabolique ».
3. Survole les cartes, clique sur le « + » de deux maladies.
4. Clique « Comparer (2) » dans le header.
5. Voit le tableau comparatif côte à côte.

## UC-3 — Découvrir une situation par pays

> Un journaliste veut visualiser le paludisme par pays.

1. Va dans `/monde`.
2. Sélectionne « Paludisme » dans le menu déroulant.
3. La choroplèthe se met à jour (Afrique colorée plus intensément).
4. Clique sur un pays → ouvre `/monde/NGA`.
5. Voit le top 5 des pays et la fiche du Nigeria.

## UC-4 — Apprendre via le quiz

> Un curieux veut tester ses connaissances.

1. Clique « Quiz ».
2. Réponds aux 8 questions générées (mix anatomie/pathologie).
3. Voit le score, la barre de progression, les explications par question.
4. Recommence avec un nouveau jeu généré.

## UC-5 — Recherche globale ⌘K

> Recherche rapide d'un terme.

1. Appuie sur ⌘K depuis n'importe quelle page.
2. Tape « cœur ».
3. Voit suggestions par type (organe, maladie, glossaire).
4. Sélectionne et navigue directement.
